yadex205-website
================

The source for www.yadex205.info


Development
-----------

### Requirements

* Node.js `>= 6.0.0` + NPM

### Prepare

```bash
$ git clone https://github.com/yadex205/yadex205-website.git
$ cd yadex205-website
$ npm install`
```

### Build website

```bash
$ npm run build
```

### Launch live preview environment

```bash
$ npm run live
```
